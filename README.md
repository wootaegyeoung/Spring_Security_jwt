# Spring_Security_jwt
각 파트별로 나눠서 spring Security와 jwt를 이용한 로그인과 회웝가입 api 서버에 대한 내용을 정리했습니다.<br>

### 버전 및 의존성
- Spring Boot 3.2.1
- Security 6.2.1
- Lombok
- Spring Data JPA - MySQL
- Gradle - Groovy
- IntelliJ Ultimate
